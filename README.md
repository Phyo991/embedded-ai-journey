# embedded-ai-journey

A hands-on journey learning embedded systems, microcontrollers, sensors, 
firmware, and robotics — through structured, practical tutorials.

## What's here

Following [Paul McWhorter's Arduino tutorial series](https://www.youtube.com/watch?v=fJWR7dBuc18&list=PLGs0VKk2DiYw-L-RibttcvK-WBZm8WLEP), 
building up embedded systems fundamentals: sensors, actuators, circuits, 
and real-time control on Arduino.

## Structure

Each folder corresponds to a lesson or group of related lessons from the series.
